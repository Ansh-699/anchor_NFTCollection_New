# Anchor NFT Collection New

This project contains an updated Anchor smart contract implementation for an NFT collection on Solana.

## Project Structure

```
anchor_NFTCollection_New/
├── migrations/
├── programs/
├── tests/
├── Anchor.toml
├── Cargo.toml
├── package.json
└── tsconfig.json
```

## Getting Started

Follow these instructions to run the project locally.

### Prerequisites

- Rust
- Solana CLI
- Anchor CLI
- Node.js

### Installation

1. Install project dependencies:

   ```bash
   yarn install
   # or
   npm install
   ```

### Usage

1. Build the program:

   ```bash
   anchor build
   ```

2. Run tests:

   ```bash
   anchor test
   ```
